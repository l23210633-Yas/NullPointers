# 🛒 Marketplace en Raspberry Pi 4 (Proyecto LAN)

## Descripción del Proyecto
Este proyecto consiste en desarrollar un marketplace tipo Amazon que funciona completamente dentro de una red local (LAN) usando una Raspberry Pi 4 como servidor principal. La plataforma no depende de la nube y funciona de manera autónoma en un evento local.

## Tecnologías Utilizadas
- Raspberry Pi 4
- Apache / Nginx
- PHP
- MariaDB
- HTML5, CSS3, JavaScript

## Objetivo del Sistema
Crear una plataforma básica de comercio electrónico para:
- Registro e inicio de sesión de usuarios
- Visualización de productos
- Carrito de compras
- Administración de productos
- Gestión de pedidos
- Base de datos local con MariaDB

## ⚙️ Estructura Inicial de la Base de Datos

**Tablas creadas:**
- **users**: Almacena la información de los usuarios registrados.
- **sellers**: Información de vendedores.
- **products**: Catálogo de productos.
- **categories**: Organización de productos por categorías (Electrónica, Tecnología, Hogar, Ropa).
- **product_images**: Gestión de imágenes de productos.
- **cart**: Carrito de compras activo.
- **cart_items**: Productos agregados al carrito.
- **orders**: Historial de compras.
- **order_items**: Detalle de productos en pedidos.
- **payments**: Información de pagos y transacciones.
- **shipping**: Seguimiento de envíos.
- **reviews**: Opiniones y calificaciones de productos.
- **wishlist**: Lista de productos favoritos.
- **notifications**: Sistema de notificaciones.
